# Base44 App


This app was created automatically by Base44.
It's a Vite+React app that communicates with the Base44 API.

Security, privacy, and compliance requirements are documented in [`docs/security_privacy_policy.md`](docs/security_privacy_policy.md).
The policy covers TLS 1.3/AES-256 encryption, key rotation, role-scoped permissions, data subject rights (FERPA/COPPA/GDPR), and AI transparency/opt-out controls.

## Running the app

```bash
npm install
npm run dev
```

## Building the app

```bash
npm run build
```
