# Teachmo App

This app is a Vite+React application.

## Running the app

```bash
npm install
npm run dev
```

## Building the app

```bash
npm run build
```

## Configuration

Set `SUPABASE_URL` and `SUPABASE_ANON_KEY` environment variables to enable Supabase features.

For more information and support, please contact the project maintainers.
