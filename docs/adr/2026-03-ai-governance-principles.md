# AI Governance and Delivery Principles

## Status
Accepted

## Context
Teachmo is evolving from AI as a feature to AI as governed infrastructure.

## Decision
Adopt platform-wide AI governance principles:
- Governance before generation
- Zero-trust data perimeter
- Progressive delivery
- Maker/checker pattern
- Policy-as-code
- Observability-first
- Stability over novelty

## Enforcement
These principles are enforced via:
- middleware
- feature flags
- required telemetry
- PR checklist
